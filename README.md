# PredicX — AI-Powered Predictive Maintenance

**PredicX** is a digital intelligence platform that uses SCADA data and AI to predict failures in power generation assets — solar panels, wind turbines, nuclear plants, oil & gas facilities, and water treatment plants — before they cause significant power loss or downtime.

---

## Live Demo

Open `index.html` in any modern browser — no build step, no dependencies.

## Tech Stack

- Pure HTML5 / CSS3 / Vanilla JavaScript — zero frameworks, zero dependencies
- CSS custom properties for full theme control
- IntersectionObserver for scroll-triggered animations
- SVG logo (inline, no external assets required)
- Photography via [Unsplash](https://unsplash.com)

## Deploying to GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`
